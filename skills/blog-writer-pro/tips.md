# Blog Writer — tips.md

## 博客运营指南

### 爆款标题公式
1. 数字型：「2024年最值得学的10个AI工具」
2. How-to型：「如何用3步搭建个人品牌」
3. 对比型：「Vue vs React，2024年该选哪个？」
4. 痛点型：「为什么你的简历总是石沉大海？」
5. 权威型：「Google工程师推荐的5个效率工具」
6. 时效型：「GPT-5发布：你需要知道的一切」

### SEO基础
- **标题**：核心关键词靠前，15-25个中文字
- **首段**：前100字包含关键词
- **H2标签**：每300-500字一个小标题
- **内链**：链接到自己其他文章
- **外链**：引用权威来源增加可信度
- **图片ALT**：描述图片内容+关键词
- **URL**：短、含关键词、用连字符分隔

### 内容日历模板
| 周 | 周二 | 周四 |
|---|------|------|
| 1 | 入门指南 | 工具推荐 |
| 2 | 深度教程 | 案例分析 |
| 3 | 观点文章 | 资源合集 |
| 4 | 行业趋势 | 读者问答 |

### 引流策略
1. **SEO长尾词**：写搜索量200-1000的长尾词文章
2. **社交分发**：每篇文章同步到3+个平台
3. **知乎回答**：回答相关问题，自然引流到博客
4. **邮件列表**：文末引导订阅
5. **社群分享**：加入相关社群，分享有价值的内容

### 变现路径
1. **广告**：Google AdSense（需要一定流量）
2. **联盟营销**：推荐工具赚佣金
3. **付费内容**：深度教程/模板/课程
4. **咨询服务**：建立专家形象后提供咨询
5. **品牌合作**：达到一定影响力后接商务合作
